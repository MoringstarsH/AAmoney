最小可行性产品描述（最终版）
1. 典型用户画像
"28岁的小李，周末和5个朋友（含一对情侣）去露营，需要按实际付款记账，但情侣作为整体分摊费用，快速算出每个小组该付多少钱。"
2. 场景故事
"六人出游含一对情侣，小王付租车费，小陈付门票（含伴侣份额），工具自动生成：小陈需给小王转180元，含情侣两人的份，小组账目两清。"
3. 功能列表
创建行程：输入行程名称并保存
管理参与人：添加参与人，可选择是个人或小组（如情侣、家庭）
记录支出：输入金额、选择付款人、选择受益人（多选）、添加备注
自动结算：实时计算每个小组应付/已付/净额，生成最优转账方案
生成截图：一键生成结算截图，包含支出明细和转账方案，可保存或分享到微信
4. 交互流程
创建行程：首页点击"新建行程" → 输入名称 → 保存进入行程主页
添加参与人：点击"添加人" → 输入姓名 → 勾选"是否为小组" → 确认添加
记录支出：点击"记一笔" → 输入金额/备注 → 选择付款人 → 确认受益人 → 保存
查看结算：点击"结算" → 查看小组应收应付 → 查看转账方案
分享截图：点击"分享" → 自动生成结算截图 → 保存到相册或分享到微信
